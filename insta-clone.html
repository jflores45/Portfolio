Instagram Clone Project

Project Overview

This project is a simplified clone of Instagram, allowing users to sign up, upload photos, follow other users, like posts, and leave comments. The goal was to replicate the core features of Instagram using modern web technologies.

Motivation/Goal

The project was inspired by the need to build a full-stack application with user authentication and CRUD operations, enhancing my skills in React and Flask. The goal was to practice creating scalable, interactive user interfaces and backends with REST APIs.

Features

  • User authentication (signup, login).
  • Upload and display photos in a user feed.
  • Follow/unfollow users and interact with posts (like, comment).
  • Responsive design for both desktop and mobile.

Technical Stack

Front-end: React, HTML, CSS, JavaScript

Back-end: Flask (Python), SQLAlchemy (Database)

Database: MySQL

APIs/Libraries: Axios, JWT (JSON Web Tokens) for authentication

Architecture/Design

The project uses a component-based architecture with React for the front-end. The data flow is managed with React state, and data is fetched from the Flask backend using REST APIs. User data and post information are stored in MySQL, and user authentication is handled with JWT.

Challenges & Solutions

One challenge was handling image uploads and displaying them efficiently. I used Cloudinary for image storage and retrieval, optimizing the file size and format to ensure quick loading times.

Lessons Learned

From this project, I learned more about managing state in React applications and improved my understanding of RESTful APIs and user authentication. I also gained experience working with third-party storage solutions like Cloudinary.

Future Improvements

Future improvements include adding real-time notifications for user interactions (e.g., likes, follows) and optimizing the feed to handle a larger number of posts more efficiently.
